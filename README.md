## Bank Buddy

### Problem

Managing personal finances and understanding banking products can be challenging for many users. There is a need for an interactive assistant that can answer banking queries, provide financial insights, and guide users through banking processes.

### Solution

Bank Buddy is an AI-powered Streamlit application that acts as a virtual banking assistant. It helps users with banking queries, provides personalized financial advice, and simplifies complex banking concepts using natural language processing.

### Architecture

- **Frontend:** Streamlit web application for user interaction.
- **Backend:** Python-based logic for handling queries and generating responses.
- **AI Model:** Integrates with language models to process and answer user questions.



### Getting Started

Follow these steps to set up your environment and run the Streamlit app.

#### 1. Create a Virtual Environment

**Windows:**
```bash
python -m venv venv
```

**macOS/Linux:**
```bash
python3 -m venv venv
```

#### 2. Activate the Virtual Environment

**Windows:**
```bash
venv\Scripts\activate
```

**macOS/Linux:**
```bash
source venv/bin/activate
```

#### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

#### 4. Run the Streamlit App

```bash
streamlit run app.py
```

For more details, visit the [GitHub repository](https://github.com/shwethaa-rajagopalan/bank-buddy).
